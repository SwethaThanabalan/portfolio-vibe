# Portfolio Website

A clean, minimal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Modern, responsive design
- TypeScript for type safety
- Tailwind CSS for styling
- Fast development with Vite
- Easy to customize

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

Edit the following files to personalize your portfolio:

- `src/components/Hero.tsx` - Update your name and introduction
- `src/components/Skills.tsx` - Add your skills
- `src/components/Writing.tsx` - Add your blog posts
- `src/components/Projects.tsx` - Showcase your projects
- `src/components/Contact.tsx` - Update contact links

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
